dotfiles
========

Use at your own risk!

```
ln -s ~/Documents/dotfiles/ ~/.dotfiles
ln -nfs ~/Documents/dotfiles/bash_profile ~/.bash_profile
ln -nfs ~/Documents/dotfiles/bashrc ~/.bashrc
ln -nfs ~/Documents/dotfiles/gemrc ~/.gemrc
ln -nfs ~/Documents/dotfiles/git-completion.bash ~/.git-completion.bash
ln -nfs ~/Documents/dotfiles/gitconfig ~/.gitconfig
ln -nfs ~/Documents/dotfiles/gitignore_global ~/.gitignore_global
ln -nfs ~/Documents/dotfiles/lldbinit ~/.lldbinit
ln -nfs ~/Documents/dotfiles/vimrc ~/.vimrc
```
